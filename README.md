# NeuS-PIR: Learning Relightable Neural Surface using Pre-Integrated Rendering

[arXiv](https://arxiv.org/abs/2306.07632)

This repository contains an efficient method for jointly learning geometry, material properties, and illumination from multi-view image observations, using pre-intergrated rendering and neural surface representation.
